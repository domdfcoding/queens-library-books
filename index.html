Queens Public Library Popular Books 2023

Dots show the locations of libraries.

The different colours indicate the same book was the most popular in those libraries.

The numbers indicate how many libraries (including that library) had that book as the most checked-out book.

White dots (with the number 1) signify books which were popular only at that library.

Click on a library for more information about that library.

For more information about popular books at Queens Public Libraries visit http://connect.queenslibrary.org/12070
